BankFlow

BankFlow es un sistema de gestión bancaria desarrollado en Java con Spring Boot. Incluye funcionalidades como transferencias, gestión de cuentas y generación de informes financieros.

Reactify

Reactify es una aplicación web moderna para la gestión de eventos. Desarrollada con React y Redux, permite a los usuarios crear, editar y compartir eventos en tiempo real.

DataCraft

DataCraft es una herramienta de análisis de datos y visualización desarrollada en Python. Utiliza bibliotecas como Pandas, Matplotlib y Seaborn para procesar y visualizar grandes volúmenes de datos.

NodeStream

NodeStream es una plataforma de streaming en tiempo real construida con Node.js y Socket.io. Permite a los usuarios transmitir y ver contenido en vivo con baja latencia.

FitTrack

FitTrack es una aplicación móvil para seguimiento de actividad física. Desarrollada en Android, incluye funciones como registro de ejercicios, seguimiento de calorías y sincronización con wearables.

BlogWave

BlogWave es un sistema de gestión de blogs personalizado desarrollado con WordPress. Incluye temas personalizados, plugins avanzados y optimización SEO.
